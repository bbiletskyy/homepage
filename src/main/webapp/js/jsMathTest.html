Let \omega be configuration of the system and \Omega be the set of all configurations. We define systems dynamics in such way that probability to find system in configuration \omega is
p(\omega)={1 \over Z} e^{-E(\omega)},
where
Z=\mathop{\sum}_{\omega \in \Omega}e^{-E(\omega)}.
After that we are to define bounded energy function E:\Omega \rightarrow \mathbb{R}, \vert E(\omega) \vert < \infty, \forall \omega \in \Omega
